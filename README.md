# Kuber Tech Solutions

Marketing site for Kuber Tech Solutions, built with React, TypeScript, Vite, Tailwind CSS, Framer Motion, and shader-based visual effects.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## Development

Start the local development server:

```bash
npm run dev
```

Open the local URL printed by Vite.
